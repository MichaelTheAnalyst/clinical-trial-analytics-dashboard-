                       üÜò EMERGENCY PROCEDURES
================================================================================

IF DASHBOARD CRASHES:
   1. Note error message
   2. Press Ctrl+C to close
   3. Run STOP_DASHBOARD.bat
   4. Wait 10 seconds
   5. Run START_DASHBOARD.bat

IF DATA LOOKS WRONG:
   1. Enable Privacy Mode OFF
   2. Check uploaded files are latest
   3. Re-upload Excel files
   4. Press F5 to refresh

IF CAN'T LOGIN:
   1. Check username: admin
   2. Check password: sctu2024 (default)
   3. Clear browser cache
   4. Restart dashboard
